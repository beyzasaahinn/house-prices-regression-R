Bu repo, Ames Housing veri seti kullanılarak yapılan çeşitli lineer regresyon analizlerini ve modelleme süreçlerini içermektedir. 
Proje kapsamında basit regresyon, çoklu regresyon, çapraz doğrulama, PLS (Partial Least Squares) ve değişken önemi analizi gibi adımlar uygulanmıştır.

# Kullanılan R Paketleri
caret — modelleme ve çapraz doğrulama
ggplot2 — görselleştirme
broom — model çıktısı düzenleme
vip — değişken önemi
pdp — kısmi bağımlılık grafikleri
